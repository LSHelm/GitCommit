## Lijst maken
Op onderstaande manier is het mogelijk om een lijst te maken. Wat mij betreft best handig!
1. test1
2. test2
    * subtest1
    * subtest2
3. test3

Ook kan er een lijst met opdrachten gemaakt worden.
- [x] Dit is gelukt.
- [ ] Dit moet nog gebeuren.
- [ ] Dit moet nog gebeuren.
